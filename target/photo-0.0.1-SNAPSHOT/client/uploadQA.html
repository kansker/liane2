Q： 我做相本需要上傳多少張照片？
  A：一般而言20頁的相本，每頁可以排 2-6張照片(平均是4)，你平均需要80張照片，完成一本相本。系統提供150MB的空間給你存JPG檔案。
Q： 我做相本印刷需要多大的檔案？
  A：一般而言在A4的版面可以排 2-6張照片，照片最少1024 x 768 像數(Pixel)就可以，一般 1280 X 960 像數也可以。
Q： 數位相機的照片標準比例是 4:3？
  A：一般數位相機的照片長與寬的比例是 4:3 為主，也就是 800 x 600 或 1024 x 768 或 1280 X 960 或 1600 X 1200。
Q： 放在編輯區的照片有點變形問題，怎麼辦？
  A：如果你的圖不是標準數位相機比例 4:3，那放到編輯框的照片就會變形，那你就要用裁切工具，修改大小。
Q： 一頁只有一張照片的，該用多大的圖檔？
  A：經過測試， 你只需要 2000 X 1600 的圖就可以，不需要更大的像數(Pixel)結果一樣好。
